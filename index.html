Tags úteis na HTML e nas CSS:


		
-Parágrafo e quebra de linha:
		 <p> e </p> para páragrafo e <br> para quebra de linha

	
-Ajustar página em qualquer dispositivo:
	<meta name="viewport" content="width=device-width,initial-scale=1.0">

	
-Exemplo de tag para deixar o CSS em um arquivo separado:
	<link rel="stylesheet" type="text/css" href="css/estilos.css">

	
-Incluir áudio no HTML e organizar em lista:
	<ul><li>
	<audio controls><source src="(CaminhoDOarquivo)/(NomeDOarquivo).mp3" type="audio/mpeg">
	Seu navegador não suporta o elemento de áudio.</audio>(NomePARAexibição)
	</li></ul>
	*Incluir <li> e </li> para cada áudio da lista
	*Incluir <ul> e </ul> para iniciar e finalizar a listagem

Exemplo de uso de tabela em HTML:
	<table>
		<tr>
			<th>Nome</th>
			<th>Sobrenome</th>
			<th>Idade</th>
		</tr>
		<tr>
			<td>João</td>
			<td>Silva</td>
			<td>30</td>
		</tr>
		<tr>
			<td>Maria</td>
			<td>Souza</td>
			<td>25</td>
		</tr>
	</table>
	Uma tabela HTML também pode incluir elementos <caption>, <colgroup>, <thead>, <tfoot> e <tbody>.
